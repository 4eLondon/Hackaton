# PlantLink – Smart Agriculture Platform

## Problem Addressed
In modern agriculture, farmers often face challenges such as limited access to real-time market information, accurate weather forecasts, AI-driven farming advice, and up-to-date farming news and due to having to rely on middle man sometimes it decreases ptofit. PlantLink addresses these issues by providing a comprehensive digital platform that connects farmers to essential tools and resources, improving productivity and decision-making.

## Why We Made It
We created PlantLink to empower farmers with technology, making agriculture more efficient and sustainable. By integrating AI assistance, marketplace access, weather updates, connect buyers to sellers directly (Messages), and news, we aim to bridge the gap between traditional farming practices and digital innovation, ultimately helping farmers increase yields and reduce risks.

## How We Made It
The project is built with a simple yet effective tech stack:
- **Frontend**: Basic HTML, CSS, and JavaScript for a responsive and user-friendly interface.
- **Database**: Supabase for secure and scalable data storage and real-time features.
- **Backend**: Node.js with Express.js to handle server-side logic and API integrations.

## Features
- **Marketplace**: Buy and sell agricultural products directly.
- **AI Assistant**: Get AI-powered advice on farming techniques and crop management.
- **Weather**: Access real-time weather forecasts tailored for farming.
- **Farming News**: Stay updated with the latest news in agriculture.
- **User Authentication**: Register and login to personalize your experience.
- **Contact**: Reach out for support or inquiries.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   cd Hackaton
   ```
2. Install backend dependencies:
   ```
   cd Backend
   npm install
   ```
3. Start the backend server:
   ```
   node script.js
   ```
4. Open `index.html` in your browser to view the frontend.

## Usage
- Navigate through the site using the navigation menu.
- Register or login to access personalized features.
- Use the AI Assistant for farming tips.
- Check the Marketplace for products.
- View Weather and News sections for updates.
